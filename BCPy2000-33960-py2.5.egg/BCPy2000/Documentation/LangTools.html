LangTools

This package is part of the evolving set of optional higher-level tools. It consists of:

  • LangTools.TreeStructure (mostly documented) provides an implementation of the trie data structure: in memory as python objects, packed in memory, or packed on disk.
  • LangTools.TextPrediction (partly documented) has implementations of various language models and a mechanism for decoding letters according to binary codebooks suitable for BCI (each symbol represented by a sequence of 1's and 0's corresponding, for example, to attended and unattended stimulus events).
  • LangTools.ModelTesting (undocumented) contains a testing framework for language models.
