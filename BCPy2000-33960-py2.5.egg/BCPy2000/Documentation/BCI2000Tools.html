BCI2000Tools

This package is part of the evolving set of optional higher-level tools. It consists of:

  • BCI2000Tools.DataFiles (partially documented) provides a convenient way of dumping arbitrary variables (lets say, the preprocessed features that you are about to classify in your online pipeline) to a file where they can be read in collated form using the and classified offline. This allows the error-prone process of offline feature extraction to be largely eliminated.
  • BCI2000Tools.FileReader (undocumented) provides a pure-python implementation of a class bcistream for reading BCI2000 .dat files.
  • BCI2000Tools.Parameters (undocumented) provides a class for representing a BCI2000 parameter and formatting it in the required format.
  • BCI2000Tools/PlaybackSourceModule.py (undocumented) contains a BciSource implementation which makes use of the FileReader module to allow .dat files to be played back through the BCPy2000 system.
  • BCI2000Tools/AudioSourceModule.py (undocumented) contains a BciSource implementation which uses the computer's sound card as the signal source. This can be useful for testing the processing of real real-time signals when you don't have access to EEG signal acquisition hardware. For best results, hook your audio output up to your audio input, and record the stimuli that are being produced by your application module.
