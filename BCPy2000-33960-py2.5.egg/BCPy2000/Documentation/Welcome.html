Welcome

Welcome to BCPy2000. If you would like to more about what this project is, you might like to read about it first.

If you'd rather just get started, you can run BCPy2000 from the .\demo directory, which contains a minimal demo installation of BCI2000, consisting of just the operator and the python-related modules.

Before you run it for the first time, you must ensure that

  1. the foundation binaries are built in .\demo\prog (they should be there already, unless you have checked the source code out from bci2000.org's subversion repository)
  2. that Python and all the necessary Python packages are installed, including the BCPy2000 framework itself.

Instructions are available if you need to install either the foundation or the framework from source.

The layout follows the usual BCI2000 convention, so the launcher scripts are in .\demo\batch. Each demo loads the necessary parameter file automatically on launch, so all you should have to do is find the operator window, click "Set Config", and then click "Start". You may also wish to read the description of the demos in order to interpret what you see.
