Presentation Phase

Definition: presentation phase

A part of an experimental trial or subtrial, defined and delineated by what stimuli are being presented, what the subject is supposed to be doing and for how long. (Examples: baseline phase, cue presentation phase, imagination phase, ...)

The BCPy2000 application module provides the infrastructure with which you can build a phase machine, which runs in a thread of its own and keeps track of when it is time to change phase. Then you only need worry about what happens at the transitions.
