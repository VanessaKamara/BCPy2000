Python

Python is a high-level, flexible programming-language designed by Guido van Rossum.

The language's design philosophy tends to lend python code a minimalist syntax, and it has a very large and comprehensive standard library. The result is a powerful, easy-to-learn and easy-to-use system which has become popular for a very wide range of applications, including scientific computing.

Our experience with python has been mostly positive. Often we embark on the implementation of a new task, write the first three lines of python, and then realize, "oh, that's it." And then it works.

The official site is http://www.python.org/.

We have had BCPy2000 working on both Python 2.4 and Python 2.5. The latter currently seems to be the most widely-adopted and -supported Python version among scientists, and is the version we recommend.

Many people are beginning to adopt Python 2.6, and our basic framework has worked under this, but since binary installers were not available for VisionEgg at the time of writing (September 2009) we have not fully tested the application module. Support for Python 3.0+, for us as for all of the projects we depend on, will probably take a while.
