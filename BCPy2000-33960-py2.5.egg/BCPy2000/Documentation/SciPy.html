SciPy

SciPy is a third-party python package for scientific computing, and is one of the most popular and actively developed free alternatives to Matlab. It is a community project sponsored and supported by Enthought.

BCPy2000 makes use of NumPy which is SciPy's underlying system for representing numerical data and performing linear algebra. While the basic BCPy2000 does not require any more of SciPy than this, you will inevitably want to install SciPy if you want to develop signal processing code.

Our optional tools make use of SciPy for constructing and applying causal filters, and for computing discrete Fourier transforms.

The project's official homepage is http://scipy.org and its documentation is here.
