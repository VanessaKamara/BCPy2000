WavTools

This package is part of the evolving set of optional higher-level tools. It consists of:

  • WavTools.Base provides a wav class, which is a convenient container class that wraps around Numpy arrays containing sound data. Various methods are implemented for editing sounds.
  • WavTools.PyAudioInterface provides a player class for playing back wav objects, and also allows recording. Recording and playback require the third-party package PyAudio.
