IPython

IPython is an interactive shell for the python programming language, written by Fernando Perez. It offers enhanced introspection, additional shell syntax, code highlighting, and tab completion. It is a component of the SciPy project, with its official homepage at http://ipython.scipy.org .

Each BCPy2000 module may optionally open an IPython shell (determined by a command-line option). The shell runs in its own thread, concurrently with the BCPy2000. From the shell, you have access to the workspace in which the module is running: the running instance is addressable as self. For example, you can type self.states to view the current values of the BCI2000 state variables, and you can manipulate their values from the command line. In the application module, you can interactively inspect and manipulate the VisionEgg stimuli while the system is running.

Aside from self, the workspace will contain very little, but you can import any packages you need, such as numpy. The framework directory and initial working directory will both be on python's search path. Your ipythonrc (which, on Windows, is usually found at
C:\Documents and Settings\%USERNAME%\_ipython\ipythonrc.ini ) can be used to perform further initialization (such as importing frequently-used packages) automatically whenever an IPython shell starts.

Tips for using the shells:

We recommend that you

  • improve the look and feel of the console windows;
  • learn to use IPython's interactive documentation system;
  • type quickref at an IPython prompt to learn more about it.
