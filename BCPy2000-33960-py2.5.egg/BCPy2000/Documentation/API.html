API

By API (or Application Programmer's Interface) we mean the methods that we have implemented, in all three modules, to help you perform certain common tasks.

Documentation for the API methods can be found in the documentation extracted from the respective framework classes:

  • BciCore
  • BciGenericSource
  • BciGenericSignalProcessing
  • BciGenericApplication

API methods have all-lower-case names, distinguishing them from hook methods which are the ones for you to implement in your subclass.

In addition to the methods, the following member attributes of self are particularly useful:

  • self.params is a dictionary of parameters, indexed by parameter name. Parameter values are stored as strings, or as lists of strings (for list parameters) or nested lists of strings (for matrix parameters). Numeric values can be obtained by the appropriate call to int or float -- for example:
    ch_ind = map(int, self.params['TransmitChList'])
  • self.states is a dictionary of the current values of the BCI2000 state variables, indexed by name and returning values of type int.
  • self.nominal is a dictionary of useful numerical values describing the setup.

These three dictionaries are "lazy" dict subclasses, meaning that elements can be accessed like .this as well as like ['this'], and tab completion is possible. Finally, some matrix parameters are defined by the BCI2000 with strings as row- and column- labels. These strings can, optionally, be used for dereferencing instead of 0-based integers. For example, type the following at one of the shell prompts after pressing 'Set Config': 

self.params.SignalSourceFilterChain
self.params.SignalSourceFilterChain.matrixlabels()
self.params.SignalSourceFilterChain[:, 'Filter Name']

Another useful feature of self.params is that, though each parameter value comes as a string, or a list or matrix of strings, the .val attribute can be used to convert it to the most suitable format for its content. For example, if you're using a full matrix as a spatial filter, then self.params.SpatialFilter.val will yield a numerical numpy.matrix.
