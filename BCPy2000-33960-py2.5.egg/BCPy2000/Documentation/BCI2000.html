BCI2000

BCI2000 is a modular software system for the recording, real-time processing and feedback of biosignals, with specific application in Brain Computer Interfacing. It is written and maintained by Gerwin Schalk and JÃ¼rgen Mellinger.

We decided to base our system on BCI2000 because it is a free, actively developed and already very comprehensive system which is fast becoming a de facto standard in BCI research. Its file format is particularly well suited for BCI research, is familiar to many researchers, and is supported by a number of cross-platform analysis tools. Its modularity makes it easy to port experiments between different signal acquisition hardware setups (of which an impressively wide range is supported). Finally, despite its unlimited reconfigurability and extensibility, its front-end can be made usable by relatively non-technical end-users.

The official site of the project is located at http://bci2000.org and includes a wiki.
