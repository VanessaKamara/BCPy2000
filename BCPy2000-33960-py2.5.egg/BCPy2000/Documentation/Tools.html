Tools

BCPy2000 comes with a set of tools which can be used as a kind of "standard library" for building experiments. These are optional: you should be able to program experiments in python, taking advantage of SciPy for signal-processing and VisionEgg and pygame for stimulus presentation, without them.

The tools are a work in progress. The current BCPy2000 release focuses on getting the framework ready, so some of the higher-level tools are not yet documented (there is no html documentation, but python docstrings exist for some items).

If you wish to use the tools in their current state, a few of them are showcased in the demo application, which may give you an idea of how they work.

Currently the tools consist of:

  • AppTools (mostly undocumented): various higher level stimuli, monitors and widgets
  • BCI2000Tools (undocumented): tools for working with BCI2000 files, and the place for any general-purpose re-useable modules such as the Playback and audio-envelope source modules.
  • LangTools (semi-documented): tools for building and testing predictive language models.
  • SigTools (semi-documented): various tools for signal processing and manipulation and classification
  • WavTools (mostly documented): tools for manipulating sounds as NumPy arrays, recording and playing them.
