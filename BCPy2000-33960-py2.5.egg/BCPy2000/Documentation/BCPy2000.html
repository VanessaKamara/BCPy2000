BCPy2000

Welcome

BCPy2000 is a system for building experimental brain-computer interface systems, based on the BCI2000 project but using Python for rapid development. Read more about the project...

News

2011-05-18: The 8th BCI2000 Workshop in Utrecht was a lot of fun, and included an extended hands-on session. In Python's home country, its power was made particularly evident by the group who, in 15 minutes flat, were already controlling the screen's background colour using a data-glove through the BCPy2000 framework.

2010-07-08: Fourth build released (v.17374). See the changelog for details.

2010-05-30: Jez repeated the C++ and Python BCI2000 tutorials at the 7th BCI2000 Workshop, May 30th-31st at the Asilomar Conference Center in California. The handout and software distro for the Python class remain the same as last time. For material similar to what I presented in the C++ class, and more, go here.

2009-10-01: Jez gave an overview and demo of BCPy2000, together with some exercises, at the 5th BCI2000 Workshop, Oct 1st-3rd at Lake George, NY. The handout specifying the exercises is available for download, for use with the new "Full Monty" software distribution (obtainable under "Option A" on the download page).

2009-09-22: Third build released (v.13637) as well as the "Full Monty" Python distribution for first-timers. See the changelog for details. We'll use this at the workshop.

2009-07-23: The CNS "Python in Neuroscience" workshop included a brief presentation of BCPy2000 and some discussions about developing python for windows.

2009-04-02: Here's a list I've compiled of Python resources for Matlab users.

2009-03-16: Second build released (v.11336). See the changelog for details.

2009-01-11: A video is available of the 10-minute BCPy2000 talk/demo given at the Machine-Learning Open-Source Software workshop, NIPS 2008.

2008-09-30: First build released (v. 8570, a.k.a. 1.0). The project is also posted on mloss.org.
