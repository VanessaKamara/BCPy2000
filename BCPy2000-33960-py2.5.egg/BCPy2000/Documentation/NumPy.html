NumPy

NumPy is a python package by Travis Oliphant. It is for representing and processing arrays of data, particularly numeric data. It forms a basis of the scientific computing environment SciPy, which is a popular and promising candidate for the eventual replacement of Matlab in scientific computing.

BCPy2000 represents a packet of signal data as a numpy.matrix object with number of rows equal to the number of different channels, and number of columns equal to the number of samples (BCI2000's SampleBlockSize parameter).

  • in_signal[m, :] addresses the (m+1)th channel across all samples, returning a matrix object
  • in_signal[:, n] addresses the (n+1)th sample across all channels, returning a matrix object
  • in_signal[m, n] addresses the (m+1)th channel, (n+1)th sample, returning a scalar float
  • numpy.mean(in_signal, axis=0) computes a common average
  • numpy.mean(in_signal.A**2, axis=1)**0.5 computes the root-mean-square across the packet, for each channel

For anything more complicated, we suggest you visit the official site and take a look at the documentation.
