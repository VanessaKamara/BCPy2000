Updating

To update your BCPy2000 framework package to a newer version:

  • If you are using the Full Monty, use the easy_install command as described on the Fully Monty documentation page. If the update package you have is an .exe file, it will not work if you simply launch it â€” use it as an input to easy_install instead.
  • If, on the other hand, you installed your existing BCPy2000 package by launching an installer .exe, then you should first uninstall the old package and then simply run the new installer. Go to the Windows control panel "Add or Remove Programs" and click on the "Change/Remove" button next to "Python 2.#.# BCPy2000 #####" (where #'s denote the appropriate version numbers). NB: from Windows Vista onwards, the "Add and Remove Programs" control panel has been renamed to "Programs and Features".
  • If you used neither easy_install nor a binary installer, then you should manually delete the Lib/site-packages/BCPy2000 directory from your Python distribution before installing the new version.
